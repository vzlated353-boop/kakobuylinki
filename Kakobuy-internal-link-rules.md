# Kakobuy Internal Linking Rules

> This document defines the internal linking structure for all 37 Kakobuy articles.
> AI must follow these rules exactly when generating articles.

## Core Linking Principles

### 1. Link Count Requirements
- **In-text links**: 2-3 links within main content (Introduction, main sections)
- **Related Guides section**: 3 links at article end
- **Total per article**: 5-6 internal links minimum
- **Maximum**: 8 internal links (avoid over-linking)

### 2. Anchor Text Rules
- **Must include target keyword**: Use the target page's primary keyword in anchor text
- **Natural integration**: Anchor text should read naturally in context
- **No generic anchors**: Avoid "click here," "read more," "this article"
- **Format**: `[Keyword-rich anchor text](relative-path-to-file.md)`

### 3. Link Placement
- **First link**: Within first 300 words (Introduction or first main section)
- **Second link**: In middle section (Benefits, Features, or How-to section)
- **Third link**: Near conclusion or in Product Highlights section
- **Related Guides**: Always at end of article

Before finalizing each article, verify:

## Common Mistakes to Avoid

—?**Over-linking**: Don't add more than 8 internal links
—?**Under-linking**: Don't add fewer than 5 internal links
—?**Duplicate links**: Don't link to same page in-text and in Related Guides
—?**Generic anchors**: Don't use "click here" or "read more"
—?**Wrong file paths**: Don't use absolute paths or wrong filename format
—?**Irrelevant links**: Don't link to unrelated pages just to meet count
—?**Missing Related Guides**: Always include exactly 3 links in Related Guides
—?**Keyword-stuffed anchors**: Keep anchor text natural, not forced
